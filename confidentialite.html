2. Données collectées et base légale

Données de bien-être (scores, mesures, notes, traitements) : stockées exclusivement dans le localStorage de votre navigateur. Aucune transmission à un serveur. Base légale : consentement (article 6.1.a RGPD).

Données d’identité (prénom, nom, date de naissance, téléphone, email) : stockées uniquement en localStorage pour personnaliser vos PDF de consultation. Jamais transmises. Base légale : consentement.

Fonctionnalité « Message à mon médecin » : lorsque vous utilisez cette fonctionnalité, un résumé anonymisé de vos scores et notes est envoyé à l’API Anthropic (Claude) pour générer un message. Aucune donnée identifiante (nom, prénom, email) n’est transmise dans cette requête. Anthropic ne conserve pas les données des appels API. Base légale : consentement (déclenché par votre action explicite).

Newsletter : si vous vous inscrivez via myboussole.fr, votre email est géré par Brevo (ex-Sendinblue), hébergé en France/UE. Désinscription possible à tout moment. Base légale : consentement.

3. Durée de conservation

Vos données locales sont conservées tant que vous ne les supprimez pas. Vous pouvez les supprimer à tout moment via Paramètres > Export/Import ou en vidant le localStorage de votre navigateur.

Les données transmises à l’API Anthropic ne sont pas conservées après le traitement de la requête.

4. Aucun tracking, aucun cookie tiers

  • Aucun cookie de suivi
  • Aucun outil d’analyse (Google Analytics, Facebook Pixel, etc.)
  • Aucune donnée partagée avec des tiers à des fins publicitaires

5. PDF générés

Les rapports PDF sont générés localement dans votre navigateur (bibliothèque jsPDF). Aucun PDF n’est transmis à un serveur.

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (UE 2016/679), vous disposez des droits suivants :

  • Droit d’accès : vos données sont directement accessibles dans l’app (Paramètres > Exporter mes données)
  • Droit de rectification : modifiez vos saisies directement dans l’app
  • Droit à l’effacement : supprimez vos données via le localStorage du navigateur
  • Droit à la portabilité : exportez vos données au format JSON depuis Paramètres

Pour exercer ces droits ou toute question : contact@myboussole.fr

7. Sécurité

L’application est servie via HTTPS (Vercel). Les données restent sur votre appareil et ne transitent par aucun serveur (sauf fonctionnalité Message médecin, cf. section 2).

8. Avertissement de santé

myBoussole est un outil de suivi personnel de bien-être. Ce n’est pas un dispositif médical. Les informations affichées ne constituent pas un diagnostic, un avis médical, ni une recommandation thérapeutique. Consultez toujours un professionnel de santé pour toute décision médicale.
